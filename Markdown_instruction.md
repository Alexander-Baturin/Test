## Списки

Для обозначения ненумерованных списков, необходимо выделить пункт звездочкой (*). Пример:
* элемент 1;
* элемент 2;
* элемент 3.

Для обозначения нумерованных списков, необходимо пункты пронумеровать с точкой (N.). Пример:
1. Первый пункт;
2. Второй пункт;
3. Третий пункт.
